### 就像 1, 2, 3 一样容易

``` bash
# 安装
npm install -g vuepress

# 创建一个 markdown 文件
echo "# Hello VuePress" > README.md

# 开始写文档
vuepress dev

# 构建静态文件
vuepress build
```
